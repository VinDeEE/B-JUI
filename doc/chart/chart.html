图表调用

本节主要介绍框架的图形报表功能（由 小策一喋 整合图表库Highcharts），Demo演示只展示了3种常用的图形报表：3D饼图、折线图及3D柱状图，其余图表样式根据对应的Hightcharts图表的json数据格式即可生成，更多图表信息请查看：Highchars中文网

界面调用：在HTML代码中放入代码：
图表数据：图表的JSON数据通过AJAX调用，data-url参数为图表JSON数据调用地址

参数（options）

名称 类型 默认值 描述
url string null [必选] 图表JSON数据调用地址
