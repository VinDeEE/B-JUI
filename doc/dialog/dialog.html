创建dialog

创建一个dialog有以下两种方式：

  • Data属性：DOM添加属性data-toggle="dialog"后，单击触发。
    DOM示例1：　打开dialog
                            打开dialog

    其他示例[加载容器中的内容]：　 

                                <button type="button" class="btn-green" data-toggle="dialog" data-id="mydialog3" data-target="#doc-dialog-target" data-title="加载容器中的内容">打开dialog</button></p>
                            <div id="doc-dialog-target" data-noinit="true" class="hide">
                                <p><input type="checkbox" id="doc-dialog-checkbox" data-toggle="icheck" data-label="测试Checkbox"></p>

                                <p><label>下拉框：</label><select data-toggle="selectpicker"><option value="1">选项一</option><option value="2">选项二</option></select></p>
                            </div>

    其他示例[回调函数]：　 
                            <script type="text/javascript">
                            var doc_dialog_test_count = 0
    
                            function doc_dialog_beforeClose($dialog) {
                                if (doc_dialog_test_count == 3) return true
                                for (var i = 0; i < 4; i++) {
                                    $dialog.alertmsg('confirm', '你确定要把我关掉？你不确定3次我是不会消失的。', {okCall: function() { doc_dialog_test_count++ }})
                                }
                                return (doc_dialog_test_count < 3 ? false : true)
                            }
                            function doc_dialog_onClose() {
                                $(this).alertmsg('info', '你刚刚关闭了一个dialog。')
                            }
                        </script>
                        <button type="button" class="btn-green" data-toggle="dialog" data-id="mydialog4" data-target="#doc-dialog-target" data-title="回调函数示例" data-before-close="doc_dialog_beforeClose" data-on-close="doc_dialog_onClose">打开dialog</button>
  • jQuery API：
    jQuery示例：　

    本例完整代码：

                        </script>
